#Scientific Calculator (Java Swing)

Description:
A Java-based GUI Scientific Calculator built using Swing, capable of performing both basic arithmetic and advanced scientific calculations. The application features a user-friendly interface, responsive buttons, and real-time computation.

Features:

Basic Operations: Addition, Subtraction, Multiplication, Division

Scientific Functions:

Trigonometric: sin, cos, tan
Logarithmic: log (base 10), ln (natural log)
Exponential: x^y, exp, x², 1/x
Other: sqrt, abs, mod, constants π and e
Utility Buttons: Clear (C), Backspace (←), Equals (=)
Error Handling: Displays “Error” for invalid operations

Technologies Used:

Java (JDK 8+)
Swing for GUI

How to Run:

Clone the repository:
git clone <repository-url>

Compile the Java file:javac ScientificCalculator.java

Run the application:java ScientificCalculator
